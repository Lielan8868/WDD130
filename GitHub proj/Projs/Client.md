<p>
The most challenging part of this project was communicating with the team and keeping the design consistent across different pages while dividing the work between team members.
</p>

<p>

</p>

<p>
If I could do the project again, I would try to improve communication and collaboration within the team. I would also ask for a clearer site framework so that everyone can follow the same structure more easily.
